# Python 编程基础

Python 是一种高级编程语言，以其简洁的语法和强大的功能而闻名。

## 变量和数据类型

Python 支持多种数据类型：
- 整数 (int): `x = 10`
- 浮点数 (float): `y = 3.14`
- 字符串 (str): `name = "Python"`
- 布尔值 (bool): `is_valid = True`

## 函数定义

在 Python 中，使用 `def` 关键字定义函数：

```python
def greet(name):
    return f"Hello, {name}!"

# 调用函数
message = greet("World")
print(message)  # 输出: Hello, World!
```

## 列表操作

列表是 Python 中最常用的数据结构之一：

```python
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0])  # 输出: apple
```
